React Native Kin SDK

RNKINSDK

Kinecosystem React Native Library (Open Source)

This is a React Native library that was built to interact with the Kinecosystem blockchain environment. It provides creating user accounts, creating transactions, getting user balance, etc. It was written with Java and JavaScript.

Check out the library on npm at react-native-kin-sdk.
