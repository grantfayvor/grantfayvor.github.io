Hexcord Media Server

HEXCORD MEDIA SERVER

Open Source Tool

Hexcord media server is an open source library for quickly deploying a full-fledged media server that can stream WebRTC to either MKV format or RTMP through FFMPEG. It was built with golang.

Check out the tool on Github @ hexcord-mediaserver.
